## 编译命令

```bash
# Go 后端 (在 Windows 交叉编译 Linux 静态二进制)
cd E:\coaether\server
GOOS=linux GOARCH=amd64 CGO_ENABLED=0 go build -o myai-server .

# 前端
cd E:\coaether\webui
npm run build    # 产物在 webui/dist/
```

## 上传方式

SCP 可能不可用（阿里云 CentOS 缺少 SFTP 子系统）。**用 SSH 管道替代**：

```bash
# 上传二进制
ssh root@8.148.182.204 'cat > /opt/coaether/myai-server && chmod +x /opt/coaether/myai-server' < myai-server

# 上传前端（tar + SSH pipe）
cd webui/dist && tar czf - . | ssh root@8.148.182.204 'rm -rf /opt/coaether/webui && mkdir -p /opt/coaether/webui && cd /opt/coaether/webui && tar xzf -'

# 上传 .env
ssh root@8.148.182.204 'cat > /opt/coaether/.env' < .env
```

## 后端自托管前端

代码修改在 `server/main.go`，加了 `NoRoute` handler：检测非 `/api`/`/ws` 路径时，先从 `webui/` 目录找静态文件，找不到则 fallback 到 `index.html`（SPA 路由）。这样单端口 8088 同时服务 API + 前端，不依赖 nginx。

## systemd 服务

```
# /etc/systemd/system/coaether.service
[Unit]
Description=Coaether Server
After=network.target

[Service]
Type=simple
WorkingDirectory=/opt/coaether
ExecStart=/opt/coaether/myai-server
Restart=on-failure
RestartSec=5
StandardOutput=file:/opt/coaether/server.log
StandardError=file:/opt/coaether/server.log

[Install]
WantedBy=multi-user.target
```

常用命令：
```bash
systemctl daemon-reload          # 修改 service 文件后
systemctl enable coaether        # 开机自启
systemctl start coaether         # 启动
systemctl restart coaether       # 重启
systemctl status coaether        # 查看状态
journalctl -u coaether -f        # 查看日志
```
