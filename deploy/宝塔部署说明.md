## 三、上传文件

宝塔面板 → 文件，进入 `/opt/coaether`（不存在则创建）：

1. 上传 `myai-server` 到 `/opt/coaether/`
2. 上传 `.env` 到 `/opt/coaether/`
3. 上传 `webui/` 整个文件夹到 `/opt/coaether/webui/`

上传后设置权限：
```bash
chmod +x /opt/coaether/myai-server
```

## 四、启动后端（进程守护）

宝塔面板 → 网站 → 没有网站类型 → 进程守护管理器（Supervisor）

如果没装 Supervisor：宝塔面板 → 软件商店 → 搜索 "Supervisor" → 安装

添加守护进程：
- 名称：coaether-server
- 启动命令：/opt/coaether/myai-server
- 运行目录：/opt/coaether
- 进程数量：1

启动后检查是否正常运行：
```bash
curl http://localhost:8088/api/health
# 应返回 {"status":"ok"}
```

## 五、配置 nginx 站点

宝塔面板 → 网站 → 添加站点（PHP版本选"纯静态"）：

- 域名填写：8.148.182.204
- 根目录：/opt/coaether/webui

然后点站点右侧的"设置" → 配置文件，替换为以下内容：

```nginx
server {
    listen 80;
    server_name 8.148.182.204;
    index index.html;
    root /opt/coaether/webui;

    # 前端静态文件
    location / {
        try_files $uri $uri/ /index.html;
    }

    # API 反代到后端
    location /api/ {
        proxy_pass http://127.0.0.1:8088;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
    }

    # WebSocket 反代
    location /ws/ {
        proxy_pass http://127.0.0.1:8088;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection "upgrade";
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }
}
```

保存后访问 `http://8.148.182.204` 应该能看到 Coaether 页面。

## 六、节点连接

当前你的 Windows 节点还在连 `localhost:8088`，需要改为连 Linux 服务器。

1. 先在 Web 端登录（首次需要注册账号），进入 节点管理 → 添加节点 → 生成一次性注册令牌
2. 在 Windows 上更新节点配置：
```bash
agent-runtime config set NODE_TOKEN <从WebUI复制的令牌>
agent-runtime config set SERVER_URL 8.148.182.204:8088
```
3. 重启 Windows 节点

## 七、防火墙检查

确保服务器防火墙放通端口：
- 80 (nginx)
- 8088 (后端，如果节点从外部连)

CentOS 防火墙：
```bash
firewall-cmd --zone=public --add-port=80/tcp --permanent
firewall-cmd --zone=public --add-port=8088/tcp --permanent
firewall-cmd --reload
```

另外宝塔面板的"安全"页面也要放行 80 和 8088 端口。
