# 用户管理

管理后台提供完整的用户生命周期管理。

## 查看用户

在「用户」页面可以看到所有注册用户，包括：

- 用户名和邮箱
- 拥有工作区数量
- 注册时间

## 查看在线用户

点击「在线」标签，只显示当前活跃的用户：

- 活跃会话数
- 在线节点数

在线用户行会以绿色高亮显示。

## 强制下线

对于异常用户或需要维护的场景，可以强制将用户踢下线：

1. 点击用户行的「下线」按钮
2. 确认操作
3. 系统会：
   - 将该用户所有运行中的会话标记为失败
   - 将该用户所有在线节点标记为离线

::: danger 注意
强制下线会中断用户正在执行的任务，请谨慎使用。
:::

## 删除用户

1. 点击「删除」按钮
2. 确认操作（不可撤销）
3. 系统会在事务中删除该用户的所有数据：
   - 节点、会话、智能体配置
   - 任务、项目、工作区
   - 评论、任务规则

::: warning 注意
不能删除自己的管理员账号，系统会阻止此操作。
:::
