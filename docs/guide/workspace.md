# 工作区

工作区是 CoAether 的核心组织单元。每个工作区有独立的智能体、任务、项目和成员。

## 创建工作区

1. 在侧边栏点击「工作区」下拉菜单
2. 选择「创建工作区」
3. 填写名称和描述

每个用户注册后会自动创建一个默认工作区。

## 邀请成员

1. 进入工作区设置
2. 点击「邀请成员」
3. 生成邀请链接并发送给团队成员
4. 新成员通过链接注册后自动加入工作区

::: warning 注意
邀请链接有过期时间，过期后需重新生成。
:::

## 成员角色

| 角色 | 权限 |
|------|------|
| **Owner** | 完全管理权限：添加/移除成员、配置智能体和任务规则、删除工作区 |
| **Worker** | 基础权限：查看和创建任务、使用智能体 |

## 工作区设置

在工作区设置中可以配置：

- **名称和描述**：修改工作区的基本信息
- **任务规则**：配置自动触发的工作流规则
- **智能体模板**：预设智能体角色供快速创建
