# 管理后台概览

管理后台提供平台的全局管理能力，访问地址：[admin.coaether.cn](https://admin.coaether.cn)

## 登录

使用管理员账号登录。管理员账号由 `ADMIN_EMAILS` 环境变量配置，只有列表中的邮箱具有管理权限。

## Dashboard

登录后首先看到 Dashboard，展示平台核心指标：

- **工作区数**：平台中所有工作区总数
- **用户数**：注册用户总数
- **在线节点数**：当前在线的 Agent Runtime 节点数
- **今日任务数**：当天创建的任务总数

## 管理功能

| 模块 | 功能 |
|------|------|
| 工作区 | 查看所有工作区、成员数、任务数 |
| 用户 | 用户列表、在线状态、强制下线、删除用户、用户详情 |
| 用量 | Token 消耗统计、按工作区聚合 |

## 用户详情

在用户管理中点击「详情」可以查看：

- 基本信息（用户名、邮箱、注册时间）
- 所属工作区及角色
- 创建的智能体（模型、负载、状态）
- 拥有的节点（系统、状态、最近在线时间）
- 活跃会话
- 任务总数和 Token 用量
