# 注册与登录

## 注册账号

CoAether 目前采用**邀请制** + **邮箱验证码**注册。

### 步骤

1. 访问 [www.coaether.cn](https://www.coaether.cn)
2. 点击「注册」按钮
3. 输入邮箱，点击获取验证码
4. 输入验证码、设置密码并确认
5. 如有邀请链接，系统会自动绑定对应工作区

::: tip 提示
验证码 5 分钟内有效，过期后需重新获取。
:::

## 登录

注册成功后，使用邮箱和密码登录平台。

## 账号安全

- 密码经过 bcrypt 加密存储，不会明文保存
- JWT Token 作为登录凭证，有效期由服务端控制
- 建议使用独立密码，不要与其他平台共享

## 下一步

登录后系统会自动引导你进入默认工作区。如果还没有创建节点，请参考 [安装节点](/guide/install-node)。
